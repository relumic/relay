# Roadmap

## V1 - Core Sync Engine *(in development)*
Priority queue with WAL-backed persistence, gRPC transport, conflict resolution (last-write-wins), chaos/disconnection test suite, single binary, Docker image.

## V2 - Visibility Layer
Web dashboard, Prometheus metrics, configurable retry and timeout policies.

## V3 - Policy Engine
Rule-based priority configuration, data TTL and drop policies, bandwidth budgeting, audit log.

## V4 - Edge AI Integration
On-device content-aware prioritization, anomaly detection on sync patterns.
